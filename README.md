Title
A Discrete Combinatorial Yamabe Flow Operator for Grayscale Image Structure Analysis: Curvature, Flow, and Conformal Enhancement

Abstract
We introduce and characterize a new discrete geometric image operator: a grayscale image is lifted to a piecewise-linear graph surface over a triangulated pixel grid and evolved by a combinatorial Yamabe flow, a per-vertex conformal rescaling that drives the discrete Gaussian curvature of the image surface toward a prescribed target field. The primary contribution of this paper is this operator and its theory, not a denoising or enhancement algorithm: we prove a curvature-singularity duality theorem showing that straight intensity edges are intrinsically flat and produce zero curvature response, so that curvature concentrates exactly at genuinely two-dimensional singularities (corners, junctions, blobs), and a monotone-energy theorem giving a Lyapunov certificate for the flow, with full proofs, and we discuss explicitly which assumptions of the classical discrete-conformal-geometry results we rely on and how (and whether) they transfer to an image-induced metric. Corner detection, structure-preserving smoothing and structure-selective sharpening are then presented as three illustrative instantiations of the same operator, each evaluated quantitatively: a corner-detection benchmark with exact synthetic ground truth (matching the Harris detector), a denoising benchmark against five established baselines including BM3D on eight standard test images, and an objective no-reference sharpness-metric evaluation of the enhancement flow. We report all of these results honestly, including where they are unfavourable: the flattening-flow denoiser is measurably better than the noisy input but trails BM3D and non-local means substantially, and we discuss why, rather than treating denoising as a competitive claim. We are explicit about a hard constraint of this study: standard large-scale benchmarks (BSD500, Kodak24, CBSD68, Set12, HPatches) were not reachable from our computing environment, so evaluation is limited to synthetic primitives, eight diverse standard test images, and one synthetic corner-detection scene; we flag this as the paper's main open limitation rather than obscuring it, and we discuss exactly what a full evaluation on those datasets would require. All code is publicly available on GitHub (link in Code Availability) and provided as supplementary material.

Keywords
discrete differential geometry, combinatorial Yamabe flow, discrete Gauss-Bonnet, image curvature, conformal image processing, structure-preserving smoothing
